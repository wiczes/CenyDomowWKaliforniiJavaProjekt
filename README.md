Analiza Rynku Nieruchomości w Kalifornii
Technologia: Java
Narzędzie do analizy danych dotyczących gospodarstw domowych. 
Główne funkcjonalności:
Sortowanie danych według parametrów technicznych (liczba sypialni).
Analiza korelacji między lokalizacją (długość geograficzna) a medianą wartości nieruchomości (zmiana procentowa).
Filtrowanie topowych rekordów z podziałem na strefy (bliskość oceanu vs. głąb lądu).
Implementacja ułatwień dostępu, obsługa klawiatury i tooltips.
Projekt zawiera testy jednostkowe.
